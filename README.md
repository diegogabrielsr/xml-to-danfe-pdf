# XML to DANFE PDF

Aplicação desenvolvida em Python para converter arquivos XML de NF-e em DANFE PDF.

## Tecnologias
- Python
- Tkinter
- NFeLib
- BrazilFiscalReport

## Funcionalidades
- Seleção de XML
- Geração automática da DANFE
- Exportação em PDF

## Como executar
pip install -r requirements.txt
python app.py
